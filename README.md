# HireSense
AI-Based Job Portal
 Developed a role-based recruitment web application supporting Admin, Employer, and Candidate workflows with
secure session management.
• Integrated a resume parsing API to extract candidate skills and implemented intelligent skill-based job matching
logic.
• Designed and optimized relational database schema and improved SQL query performance for faster job filtering and
retrieval.
• Implemented server-side validation, exception handling, and modular MVC architecture to enhance backend
reliability and scalability.
